# menyoo汉化项目
现在中文社区出现的menyoo汉化都是通过翻译软件更改menyoo.asi内部字符来进行的汉化，并且都会带有个人的印记。Menyoo从1.8.0版本开始就已经更新了外置汉化功能，就是可以让玩家自己编辑作者提供的专有可编辑语言文件对修改器里面的词语进行翻译，比如雷霆修改器的外置翻译是个xml格式的文件。
但是作者提供的语言文件示例的字符（在language文件夹中）仅仅只有修改器庞大词库的5%左右（个人估算），并且许多字符如果不通过第三方软件是无法找到准确的原始字符，比如游戏里面你看到的是词语是123，但是实际字符可能是~r~123（必须找到准确的原始字符，你的翻译才会被读取），因此我通过软件将修改器内的字符全部提取转换成修改器提供的语言文件一样的格式，再通过谷歌机翻了所有，润色了一部分然后上传到此。
